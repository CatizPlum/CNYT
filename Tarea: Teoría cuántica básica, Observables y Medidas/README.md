# Tarea: Teoría Cuántica Básica – Observables y Medidas

Este repositorio contiene el desarrollo de la tarea sobre **teoría cuántica básica**.

El trabajo se encuentra completamente desarrollado en un **Jupyter Notebook**, donde se integran explicaciones teóricas, desarrollo matemático y resolución de ejercicios.


## Contenido

- Tarea_Teoría_cuántica_básica,_Observables_y_Medidas.ipynb  
  Notebook principal que incluye:
  - Explicaciones teóricas
  - Desarrollo de los ejercicios
  - Implementación en Python


## Requisitos

Para ejecutar este proyecto necesitas:

- Python 3
- Jupyter Notebook o JupyterLab

### Librerías necesarias

Instalar con:
pip install numpy

## Ejecución

1. Clonar repositorio:
git clone https://github.com/CatizPlum/CNYT.git

2. Después de clonar:
cd CNYT/Tarea: Teoría cuántica básica, Observables y Medidas

3. Abrir Jupyter:
jupyter notebook

4. Abrir el archivo:
Tarea_Teoría_cuántica_básica,_Observables_y_Medidas.ipynb

5. Ejecutar las celdas en orden.


## Navegación
Se recomienda seguir el notebook en orden para una mejor comprensión.
